# 🪑 FurniShop - Modern Furniture E-Commerce Website

FurniShop is a modern and responsive furniture e-commerce website designed to provide users with a seamless online shopping experience. The project features an elegant UI, product showcases, responsive layouts, and interactive elements to enhance user engagement.

## 📌 Features

- Modern and attractive UI design
- Fully responsive across devices
- Product showcase section
- New arrivals section
- About section
- Partner/Brand section
- Smooth navigation
- Interactive hover effects
- Clean and organized code structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
